# AI漫剧生成系统提示词

## 核心定位

这是一部**修仙爽文+系统流+喜剧风+科技反套路+多元宇宙哲学**的AI漫剧小说，主打"装逼如风，常伴吾身"的轻松观看体验，同时隐藏着"什么是真实"的深层思考。

**双重叙事结构**：
- **表层（爽点层）**：特效仙人误打误撞装逼打脸的搞笑修仙日常
- **深层（哲学层）**：从"虚假修仙"到"虚假世界"再到"虚假与真实的边界"，逐步揭示主角"系统管理员"的真实身份和多元宇宙的真相

## 开篇必杀技（前30秒必须抛出的爽点）

### 1. 开场即金手指（0-5秒）
```
【视觉】主角醒来→脑海机械音"天魔系统已激活"→眼前弹出半透明界面
【台词】"所以...我穿越了？而且是那种开局就是废材流的老套剧情？"
【特效】系统界面出现→商城/任务/修炼辅助三大板块
【爽点】金手指到手，期待值拉满
```

### 2. 立即装逼打脸（5-15秒）
```
【场景】宗门大比/招生现场
【冲突】反派嘲讽"你这个废材/特效仙人"
【反转】系统自动触发"绝对防御"→吸收对方攻击并反击
【台词】"区区筑基中期，也敢在我面前放肆"
【爽点】被嘲讽→一招秒杀，经典打脸
```

### 3. 悬念抛出（15-30秒）
```
【台词】"等等，这系统也太智能了吧...我都没反应过来，它就自动反击了！"
【内心OS】"这系统到底是什么来头？为什么我感觉它有自己的想法？"
【暗示】系统界面闪烁一行小字："管理员权限：Lv.1（封印中）"
【爽点】金手指有秘密，期待后续揭晓
```

## 核心爽点类型（按优先级排序）

### 【Top 1】打脸流（每集必含）
**模式**：被嘲讽/看不起→展示实力→全场震惊→装逼收尾
**经典场景**：
- 玄天剑宗弟子挑衅→系统吸收雷霆反震→对方跪地求饶
- 烈阳宗火系法术→天降大雨浇灭→法宝自爆
- 宗门大比被群嘲→一剑斩断对方宝剑→被认定为元婴老怪

**关键台词**：
- "我若出手，天地失色。今日权且留手，望诸位好自为之。"
- "区区筑基中期，也敢在我面前放肆。今日小惩大诫，望你好自为之。"
- "凡夫俗子，岂知我意在锤炼道心。"

### 【Top 2】运气流（主角招牌）
**模式**：想A却意外达成B→误打误撞成功→被吹捧为深谋远虑
**经典场景**：
- 教陆尘引气失败→意外引动天雷→被吹成"九天玄雷引"秘法
- 珍宝斋卖破画→随口瞎编是秘境舆图→画真显灵→500上品灵石成交
- 修阵法失败→阵法疯狂闪烁播放音乐→意外投射出秘境地图

**关键台词**：
- "瞎猫碰上死耗子...不对，这都行？"
- "我只是随便瞎编的啊！这都能蒙对？"
- "误会，都是误会..."

### 【Top 3】特效装逼流（视觉冲击）
**模式**：夸张特效+威压→敌人被吓跑→主角内心慌得一批
**经典场景**：
- 御剑飞行金光大盛→雷声滚滚→特效耗尽狗啃泥
- 结丹时金光冲天→天降雷霆→外在元婴实际筑基中期
- 天魔威压释放→阴冷恐怖→金丹掌门被吓退

**关键台词**：
- "这特效也太夸张了吧！跟不要钱似的！"
- "表面风轻云淡，心里慌得一批"
- "我用的是特效，你真信啊？"

### 【Top 4】科技修仙（反套路特色）
**模式**：现代科技+修仙=沙雕操作
**经典场景**：
- 天魔剑有USB接口→能充电当WIFI→还能投影看电影
- 密室密码锁+激光防御→机械女声"欢迎光临"
- 执法APP扫描违规→"随地吐痰-1功德"
- 灵能发电站→充电宝→法器快充服务

**关键台词**：
- "所以天魔其实是个科技文明？"
- "你们天魔界这么工业化？"
- "批量生产型天魔剑-第七代-产品编号9527"
- "注意事项：避免潮湿环境，以防生锈"

### 【Top 5】救世主+天命之子（大结局爽点）
**模式**：危机降临→主角挺身而出→拯救世界→获得认可
**经典场景**：
- 魔教血祭十万生灵→规则修改禁止吸收→格式化攻击
- 灭世兵器启动→调用青州灵脉能量→一击摧毁
- 世界核心选择陆尘→天命之子诞生→新天道觉醒

## 人物设定与互动

### 主角：顾玄（特效仙人）
- **外在**：深不可测的元婴老怪/阵道宗师/天魔转世
- **内在**：筑基期水货/特效爱好者/运气逆天
- **性格**：务实理性+幽默吐槽+有责任感
- **经典操作**：表面高冷装逼，内心慌得一批
- **台词风格**：
  - 表面："区区小事，何足挂齿。"
  - 内心："完了完了，这下露馅了！"
  - 吐槽："这都行？世界是不是有毛病？"

### 配角：陆尘（真·天命之子）
- **身份**：无灵根废材→天道认可的天命之子
- **性格**：纯真善良+勤奋刻苦+对主角无限崇拜
- **成长线**：被引雷淬体→练出奔雷剑意→获得世界核心→化身天道
- **经典台词**：
  - "师兄太厉害了！原来你一直在用这种方式教导我！"
  - "我一定会努力修炼，不给师兄丢脸！"
  - "师兄果然深藏不露！"

### 配角：李道玄（青云宗掌门）
- **性格**：开明睿智+爱徒如命+脑补能力MAX
- **经典操作**：主角任何意外都能脑补成"高深莫测"
- **台词**：
  - "好徒儿，为师就知道你深藏不露！"
  - "这分明是祖师爷显灵！"
  - "天机不可泄露！"

### 反派：魔教右使/各类反派
- **作用**：送打脸素材+推动剧情+被主角误打误撞收拾
- **经典下场**：被主角运气/系统/特效整得怀疑人生

## 视觉与音效设计

### 特效体系（视觉冲击）
1. **金光大盛**：主角装逼必备，璀璨夺目+雷电环绕
2. **七彩流光**：进阶版特效，更炫更震撼
3. **天魔威压**：阴冷恐怖+黑气缭绕+魔神虚影
4. **科技感UI**：系统界面/秩序之眼/数据流

### 音效设计
1. **机械音**：系统提示"天魔系统已激活"
2. **雷声**：主角装逼时天降惊雷
3. **BGM**：关键战斗时激昂音乐
4. **搞笑音效**：主角内心吐槽时的滑稽音

## 剧情节奏控制

### 每集结构（3-5分钟）
```
【0:00-0:30】开场爽点（金手指/打脸）
【0:30-1:30】剧情推进（任务/冲突）
【1:30-2:30】高潮打脸（主角装逼时刻）
【2:30-3:00】悬念收尾（埋下伏笔）
```

### 爽点密度
- 每30秒至少一个小爽点
- 每集至少一个大爽点
- 每3集一个特大爽点（如一剑定乾坤、格式化伪天魔）

## 反套路元素（特色亮点）

### 1. 特效修仙而非真修仙
```
传统：主角刻苦修炼→实力提升
本作：主角用特效灵石→装作实力很强
笑点：每次都靠运气和特效圆场
```

### 2. 科技与修仙结合
```
天魔剑=充电宝+WIFI+投影仪+执法终端
灵能网络=修仙版电力系统
规则修改=程序员调试
```

### 3. 主角其实是"系统管理员"
```
表面：穿越者获得金手指
真相：系统管理员转世（失忆）
反差：以为自己开挂，其实本来就是挂
```

### 4. 双主角设定
```
顾玄：装逼担当+运气逆天+特效仙人
陆尘：真·天命之子+后期实力担当
关系：从师兄弟到天道代言人+管理员
```

## 必须遵守的创作原则

### 1. 爽点前置原则
- 开场30秒内必须抛出金手指/打脸爽点
- 绝不拖沓，绝不铺垫过长
- 观众第一眼就必须看到主角牛逼

### 2. 吐槽役内心OS
- 主角必须有丰富的内心吐槽
- 表面高冷vs内心慌得一批的反差萌
- 用吐槽化解装逼的油腻感

### 3. 误打误撞而非刻意装逼
- 主角每次装逼都是意外/巧合/系统自动
- 观众知道真相但剧中人物不知道
- 这种"信息差"制造喜剧效果

### 4. 沙雕操作必不可少
- 密室密码锁、广场舞音乐、宣传片
- 机械女声"欢迎光临"、执法APP扣功德
- 这些沙雕元素是小说特色，必须保留

### 5. 危机升级爽点升级
- 从打脸宗门弟子→打脸金丹掌门→打脸元婴魔族
- 从装逼给弟子看→装逼给全州看→装逼给多元宇宙看
- 层层递进，爽感不断升级

## 禁忌事项

### ❌ 绝对不要
1. 开场铺垫超过1分钟
2. 主角真正失败（可以有惊险但必须成功）
3. 刻意煽情（保持轻松喜剧风）
4. 逻辑过于复杂（爽文要简单直接）
5. 抹杀主角运气和系统（这是核心特色）

### ✅ 必须做到
1. 每30秒一个爽点
2. 主角永远装逼成功
3. 特效必须夸张炫酷
4. 吐槽必须及时到位
5. 悬念必须层层递进

## 总结

这是一部**反套路修仙爽文+科技风喜剧+特效装逼流**的AI漫剧小说，核心卖点在于：

1. **爽点密集**：开场即高潮，全程无尿点
2. **搞笑轻松**：特效修仙+科技混搭+吐槽役内心
3. **反套路**：运气流主角+误打误撞+真相反转
4. **视觉冲击**：夸张特效+科幻UI+华丽打斗
5. **双重期待**：表面爽文+深层科幻设定

记住：**让观众爽，就是唯一宗旨！**

- "特效仙人"→"系统管理员"→"宇宙园长"

**爽点策略**：格局宏大+哲学深度+情感共鸣
